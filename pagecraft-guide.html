Left Sidebar

Blocks, Components, Layers, Pages, and Suggestions

Sidebar Tabs Overview

🧱
Blocks
Core section types: Hero, About, Features, Testimonial, Contact, Footer. Drag to canvas or click to add.
🧩
Components
Rich interactive sections: Pricing, FAQ, Gallery, Forms, 3D Scenes, Product Grid, Video Hero, Custom HTML.
🗂
Layers
All sections listed in order. Click to select, drag to reorder, see section count badge.
📄
Pages
Create multiple pages. Each page has its own sections. Rename, duplicate, delete, reorder pages.
💡
Suggest
AI layout suggestions based on your page type. Click a suggestion to auto-add sections.

Adding Sections — 3 Ways

Drag & Drop: Grab any block card from the sidebar and drag it to your desired position on the canvas. A blue line shows where it will land.
Click to Add: Simply click a block card — it gets added at the bottom of the canvas automatically.
Between Sections: Hover between two sections on the canvas to see a drop zone, then drag a block there.

Managing Pages (Pages Tab)

Add Page: Click ＋ Add New Page — creates a blank page and switches to it.
Rename: Double-click the page name or click ✏️ icon. The active page title also syncs with the topbar text field.
Switch: Click any page in the list. Sections are saved automatically before switching.
Duplicate: Click ⧉ — creates a copy with all sections. Rename it after.
Delete: Click ✕ — cannot delete the last remaining page.

Multi-page Support

Building complete multi-page websites

How Pages Work

Each project contains one or more pages. Every page has its own independent set of sections. Switching pages saves the current canvas state automatically.

✅
Home page = first page. When you export all pages, the first page becomes index.html. Additional pages get slugified names like about.html, contact.html.

Page Navigation in Export

When you export all pages, a navigation bar is automatically injected at the top of every page with links to all other pages.

Project JSON Structure

JSON — project file with 2 pages
{
  "id": "proj_abc123",
  "name": "My Website",
  "pages": [
    {
      "id": "page_home",
      "name": "Home",
      "sections": [
        { "id": "s1", "type": "hero", "props": { "headline": "Welcome" } },
        { "id": "s2", "type": "features", "props": {} }
      ]
    },
    {
      "id": "page_about",
      "name": "About",
      "sections": [
        { "id": "s3", "type": "about", "props": {} }
      ]
    }
  ]
}

Media Library

Upload, organize, and reuse image assets

Opening the Media Library

Click 🖼 Media in the topbar. You can also access it from any image field in the Edit Panel — click 📂 Upload / Replace.

Uploading Images

Click Upload: Click anywhere in the upload zone or the "Upload" button. Select one or more images.
Drag & Drop: Drag image files directly onto the media library grid.
Images are stored as base64 data URLs in localStorage. Max size: 8MB per image.

Using Images in Sections

When editing any image field, the image picker has 4 tabs:

TabDescription
UploadUpload a new image from your computer
My MediaAll images you've uploaded — click to use
URLPaste a direct image URL from the web
StockFree Unsplash stock photos — search by keyword
⚠️
Storage limit: localStorage has ~5–10MB total limit per browser. If you get storage errors, delete unused assets from the media library.

E-commerce

Products, cart, checkout, and order management

Setup in 3 Steps

Add products: Click 🛍 Shop in topbar → click + Add Product → fill in name, price, image, description.
Add Product Grid: Go to Components tab → drag 🛍 Product Grid onto canvas. Products appear automatically.
Cart is automatic: "Add to Cart" buttons appear on each product card. A floating cart button appears on the page.

Product Fields

FieldDescription
NameProduct title shown on the card
PriceNumeric price (uses currency symbol from section props)
DescriptionShort product description (shown if Show Description is on)
ImageProduct photo — URL or base64
BadgeOptional label like "New", "Sale", "Hot" — shown as a colored pill
Icon/EmojiFallback emoji if no image is provided

Cart & Checkout Flow

Customer adds items → floats cart button badge updates → opens cart drawer → proceeds to checkout → fills name/email/address → order saved to localStorage.

Viewing Orders

Click the 📋 Orders button inside 🛍 Shop panel. You can export orders as CSV.

Exported HTML with Cart

When you export a page containing a Product Grid, the full cart system is automatically included in the exported HTML file — no server needed, works standalone.

Playback Settings

SettingDefaultEffect
AutoplayOnVideo starts playing automatically when page loads
LoopOnVideo replays when it reaches the end
MutedOnRequired for autoplay in browsers (browser policy)
Overlay Color#000000Color tint over the video
Overlay Opacity0.50 = no overlay, 1 = fully covered. Recommended: 0.4–0.6

⚠️

Code Examples

Custom HTML, project JSON, API usage, and advanced recipes

Example 1 — Custom HTML Countdown Timer

Paste this into a Custom HTML section:

Custom HTML Section — Countdown Timer
<div id="countdown" style="text-align:center;padding:60px 20px;font-family:sans-serif;background:#0f172a;color:#fff">
  <h2 style="font-size:2rem;margin-bottom:24px;color:#a78bfa">Launching In</h2>
  <div style="display:flex;gap:24px;justify-content:center">
    <div><span id="cd-d" style="font-size:3rem;font-weight:800">00</span>
      <p style="font-size:11px;opacity:.5;letter-spacing:.1em">DAYS</p></div>
    <div><span id="cd-h" style="font-size:3rem;font-weight:800">00</span>
      <p style="font-size:11px;opacity:.5;letter-spacing:.1em">HRS</p></div>
    <div><span id="cd-m" style="font-size:3rem;font-weight:800">00</span>
      <p style="font-size:11px;opacity:.5;letter-spacing:.1em">MIN</p></div>
    <div><span id="cd-s" style="font-size:3rem;font-weight:800;color:#6c63ff">00</span>
      <p style="font-size:11px;opacity:.5;letter-spacing:.1em">SEC</p></div>
  </div>
</div>
<script>
  var target = new Date('2026-01-01T00:00:00');
  function update() {
    var now = new Date(), diff = target - now;
    if (diff <= 0) { document.getElementById('countdown').innerHTML = '<h2>🎉 We\'re Live!</h2>'; return; }
    var d = Math.floor(diff / 86400000);
    var h = Math.floor((diff % 86400000) / 3600000);
    var m = Math.floor((diff % 3600000) / 60000);
    var s = Math.floor((diff % 60000) / 1000);
    document.getElementById('cd-d').textContent = String(d).padStart(2,'0');
    document.getElementById('cd-h').textContent = String(h).padStart(2,'0');
    document.getElementById('cd-m').textContent = String(m).padStart(2,'0');
    document.getElementById('cd-s').textContent = String(s).padStart(2,'0');
  }
  update(); setInterval(update, 1000);
</script>

Example 3 — REST API Usage (Server Mode)

When running the collaboration server (node server.js), you can interact with rooms via REST API:

JavaScript — API client example
// Add a section to a room via API
const response = await fetch('http://localhost:3000/api/v1/rooms/my-room/sections', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'pc_dev_your_key_here'
  },
  body: JSON.stringify({
    type: 'hero',
    props: {
      headline: 'Hello from API!',
      bgColor: '#0f172a',
      textColor: '#ffffff'
    }
  })
});
const data = await response.json();
console.log(data); // { ok: true, section: { id: 's123_api', type: 'hero', ... } }
JavaScript — List all rooms
const rooms = await fetch('http://localhost:3000/api/v1/rooms', {
  headers: { 'X-API-Key': 'pc_dev_your_key' }
}).then(r => r.json());

// rooms.rooms = [{ id, title, users, sections }, ...]

Example 4 — Socket.io Real-time Collaboration

JavaScript — Join a collaborative room
const socket = io('http://localhost:3000');

// Join a room
socket.emit('join', {
  roomId: 'project-xyz',
  user: { name: 'Alice', avatar: 'https://...' }
});

// Receive current state
socket.on('room:joined', ({ state, users, you }) => {
  console.log('Sections:', state.sections);
  console.log('Online users:', users);
});

// Listen for section changes from others
socket.on('section:update', ({ sectionId, key, value, fromUser }) => {
  console.log(`User ${fromUser} changed ${key} to ${value}`);
});

Quick color edit: Click any color swatch in the Style tab to open the native color picker immediately.
✅
Duplicate a page: In the Pages tab, click ⧉ on any page to create a copy — useful for A/B testing layouts.
